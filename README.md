### GLTFUtility
Allows you to import and export GTLF files during runtime and in editor.

**NOTICE** This is a work in progress. Current features can be tracked below:

- [x] Editor import
- [ ] Editor export
- [ ] Runtime import API
- [ ] Runtime export API
- [x] Static mesh
- [x] UVs
- [x] Normals
- [x] Tangents
- [x] Vertex colors
- [ ] Materials
- [ ] Textures
- [x] Rig (Has issues)
- [ ] Animations
- [ ] Morph targets
- [x] GLTF format
- [ ] GLB format
